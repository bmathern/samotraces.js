Class: ObselInspector

Samotraces.UI.Widgets. ObselInspector

Widget for visualising an Obsel as an HTML list.

new ObselInspector(html_id, obsel_selector)

Widget for visualising an Obsel as an HTML list.

Samotraces.UI.Widgets.ObselInspector is a generic Widget to visualise Obsels.

This widget observes a Selector object. When an obsel is selected, the information about this obsel is displayed in the widget. When an obsel is unselected, the widget closes. Clicking on the red cross will close the widget (and automatically unselect the obsel). When no obsel are selected, the widget is not visible, selecting an obsel will make it appear.

Parameters:
Name Type Description
html_id String

Id of the DIV element where the widget will be instantiated
obsel_selector Selector.<Obsel>

A Selector of Obsel to observe.
